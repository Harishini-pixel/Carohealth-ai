# Healthcare Discharge Instruction Simplifier

An AI-powered healthcare assistant that simplifies discharge instructions, explains medicines, tracks patient progress, and visually indicates health risk using a five-color safety system.

## Features
- Simplified discharge explanation
- Tablet explanation
- Five-color health safety system (Green → Black)
- Daily care guidance & reminders
- Progress tracking (weight, height, BMI)
- Multilingual support
- Report export & offline access
- Clear medical disclaimer

⚠️ This application does NOT provide medical diagnosis.
